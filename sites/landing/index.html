Nginx Docker Lab

Une plateforme de demo complete pour Nginx.

Cette stack expose un reverse proxy HTTPS, du load balancing, du cache, du rate limiting, une zone protegee par mot de passe, un second site statique et une couche d'observabilite optionnelle avec Prometheus et Grafana.

Tester le backend equilibre Voir l'API proxifiee Verifier le cache Zone protegee
Reverse proxy

Les routes /app/ et /api/ sont transferees a des services Docker internes.

Load balancing

Nginx distribue les requetes entre app1 et app2 avec la strategie least_conn.

Securite

HTTPS force, en-tetes de securite, masquage des versions et basic auth sur la zone d'administration.

Observabilite

Le endpoint /nginx_status alimente un exporter Prometheus active via un profil Compose.
